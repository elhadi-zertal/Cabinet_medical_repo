# Cabinet Medical Management System

## Description
The **Medical Office Management System** is designed to assist medical offices in managing appointments and patient records. This system simplifies administrative tasks such as scheduling appointments and tracking patient data.

## Features
- Manage patient appointments
- Track and update patient records
- Handle various medical office administrative tasks

## Technologies Used
- Java

## Installation Instructions
1. Clone the repository to your local machine.
2. Navigate to the following directory:
CabinetMedical/src/main
3. Open the project in your preferred IDE (e.g., IntelliJ IDEA, Eclipse).
4. Locate the `App.java` file inside the `EXE` folder.
5. Run the `App.java` file to launch the application.

## Usage
- Once the application is running, follow the on-screen instructions to manage appointments and patient records.
